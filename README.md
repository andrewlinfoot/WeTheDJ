WeTheDJ
=======
A platform that allows you to collaborativly DJ with other Smartfile Users in real time.

1. Create a 'wethedj' folder in your Smartfile root and uplaod .mp3 files
2. Login with Smartfile
3. DJ and Enjoy

This applicaiton was built using the [Meteor.js](http://meteor.com "Meteor.js") framework and is currently running on Meteor version 0.5.9.
